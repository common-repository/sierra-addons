Plugin with extra features for [WP Sierra](https://wpsierra.com/) theme.

== Description ==

Sierra Addons add features to [WP Sierra](https://wpsierra.com/) theme:
- meta boxes which gives you the ability to set pages transparent headers and alternative logo in header
- Elementor widgets like:
  * WP Sierra Button
  * WP Sierra Posts
- WP Sierra demo sites


IMPORTANT: this plugin requires [WP Sierra](https://wpsierra.com/) theme to be installed to work.

== Installation ==

1. Upload sierra-addons to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. That's all.


== Frequently Asked Questions ==

= Does not have all features? =

This plugin requires [WP Sierra](https://wpsierra.com/) theme to have full functionality.
